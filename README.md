## What I Build

### 🔐 Infrastructure & Authentication

**PlayerAuth** is a custom access-control plugin for Minecraft networks running on Velocity proxy. The whole network operates with `online-mode=false`, but behaves like a premium-only setup. Premium users join without friction. Cracked users get blocked unless they're on an explicit whitelist — and even then, they go through a manual authentication flow.

It handles packet-level authentication, session management, and the hybrid premium/cracked access logic. Built specifically to survive Minecraft version upgrades with minimal changes. This is running in production at [UnstableLeaks](https://discord.gg/unstableleaks), where I lead development.

> **Stack:** Java, Velocity Proxy API, Purpur

<details>
<summary><b>Why this matters</b></summary>
<br/>
Running <code>online-mode=false</code> is how most cracked servers operate. But I built a system that makes a fully offline-mode network behave like a premium-only service. Premium players never notice anything different. Cracked players get rejected unless explicitly whitelisted — and those whitelisted accounts authenticate through a manual verification flow. This solves the biggest problem Minecraft networks face: protecting the experience for legitimate premium users while offering controlled access for specific non-premium accounts. All without touching online-mode.
</details>

## Open Source

### PocketUI — Form Library for PocketMine-MP
A fluent API for building interactive UIs in Minecraft Bedrock servers. Supports Simple Forms (button menus), Modal Forms (dialogs), and Custom Forms (inputs, sliders, toggles), all with chained callbacks.

```
⭐ 1  |  🍴 1  |  PHP  |  Apache-2.0
```

### GraveLoot — Death Chests Plugin
Smart death chest system for PocketMine-MP. Creates single or double chests on player death based on how much inventory they had, with glowing particle guides and configurable timers. Survives explosions and cleans itself up.

```
⭐ 0  |  🍴 0  |  PHP  |  Apache-2.0
```
