## What I Build

### 🔐 Infrastructure & Authentication
**PlayerAuth** — Custom access-control plugin for Minecraft networks running Velocity proxy. Handles packet-level authentication, session management, and hybrid premium/cracked access with whitelist override. Operates with `online-mode=false` while behaving like a premium-only network. Built for maintainability and version-upgrade resilience across future Minecraft releases.

> **Stack:** Java, Velocity Proxy API, Purpur · [discord.gg/unstableleaks](https://discord.gg/unstableleaks)

<table><tr><td>
<details>
<summary><b>Why this matters</b></summary>
<br/>
Most Minecraft networks either run fully offline (no auth) or fully online (no cracked users). PlayerAuth bridges both — premium users pass through seamlessly, cracked users hit a manual authentication gate. Only whitelisted cracked accounts get through. This solves the fundamental pain point for network owners: protecting against unauthorized access without blocking legitimate players who can't afford a premium account.
</details>
</td></tr></table>

## Open Source

### PocketUI — Form Library for PocketMine-MP
Fluent API for building interactive UIs in Minecraft Bedrock servers. Supports Simple Forms (button menus), Modal Forms (dialogs), and Custom Forms (inputs, sliders, toggles) with chained callbacks.

```
⭐ 1  |  🍴 1  |  PHP  |  Apache-2.0
```

### GraveLoot — Death Chests Plugin
Smart death chest system for PocketMine-MP. Auto-creates single/double chests based on inventory size with glowing particle guides and configurable timers. Survival-proof: chests survive explosions and auto-cleanup.

```
⭐ 0  |  🍴 0  |  PHP  |  Apache-2.0
```
